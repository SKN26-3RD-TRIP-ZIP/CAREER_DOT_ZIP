# 1. 프로젝트 개요

## 📌 프로젝트 소개

기존 모의면접 서비스는 정적인 질문을 반복적으로 제공하거나, 지원자의 경험과 지원 직무를 충분히 반영하지 못하는 경우가 많습니다. 또한 면접 이후 자신의 답변을 객관적으로 분석하고 보완할 수 있는 피드백이 부족하여, 실질적인 면접 역량 향상으로 이어지기 어려운 한계가 있습니다.

**CAREER.zip**은 이러한 문제를 해결하기 위해 개발된 **AI 기반 IT 취업 맞춤형 모의면접 코칭 플랫폼**입니다.

사용자가 업로드한 **이력서, 자기소개서, 채용공고(JD), GitHub 프로젝트**를 종합적으로 분석하여 지원자의 경험과 지원 직무에 적합한 면접 질문을 생성합니다. 이후 답변 내용을 AI가 분석하여 평가하고, 부족한 부분은 꼬리질문으로 심층 확인하며, 면접 종료 후에는 종합 리포트를 제공하여 지속적인 성장을 지원합니다.

# 6. AI 면접 프로세스

## 🔄 전체 면접 흐름

```text
지원 자료 등록
(이력서 · 자기소개서 · JD · GitHub)
                │
                ▼
        지원 자료 분석
                │
                ▼
      AI 맞춤형 질문 생성
                │
                ▼
         사용자 답변 입력
        (텍스트 / 음성)
                │
                ▼
         AI 답변 품질 평가
                │
                ▼
      답변 충분성 여부 판단
                │
      ┌─────────┴─────────┐
      │                   │
      ▼                   ▼
 꼬리질문 생성         다음 질문 진행
      │                   │
      └─────────┬─────────┘
                ▼
        면접 종료 및 분석
                │
                ▼
       최종 면접 리포트 생성
```

## 📋 평가 기준

사용자의 답변은 다음 네 가지 기준을 중심으로 평가됩니다.

| 평가 항목 | 설명 |
|-----------|------|
| 전문성 | 기술 및 직무 관련 내용을 정확하게 설명했는가 |
| 논리성 | 답변이 일관된 흐름으로 구성되어 있는가 |
| 구체성 | 경험과 근거를 충분히 제시했는가 |
| 전달력 | 핵심 내용을 명확하게 전달했는가 |
